# Honours College Project: Food Collector

## Duration and Workload
This project will run from 1 November 2023 to 31 May 2024 (7 months). There will be 15 EC awarded for the project, corresponding to 420 hours. This amounts to 60 hours per month, or approximately 15 hours per week.

## High Level Research Goals
1. Implement an agent-based foraging paradigm in software allowing for manipulation of neural network topology.
2. Implement different foraging scenarios of various levels of complexity.

## Specifications
The software implementation needs to allow for a Python-based neural controller. Examples are PyTorch or TensorFlow, but this is not a strict requirement. It is important that controllers should be interchangeable: architecture should be modular so that different controllers can be tried out.
